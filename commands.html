Command List

  • Hello World (or hw)
    • Says aloud "Hello World" using mac's 'say' command
  • Open {url}
    • Opens {url} in web browser
  • Say {message}
    • Says aloud {message} using mac's 'say' command
  • Picture
    • Takes a picture using the computer's camera and sends it back
  • Screenshot
    • Takes a screenshot of the computer's screen and sends it back
  • Play {platform}
    • Runs {platform} music platform, playing whatever song is queued
  • Set Volume {number}
    • Sets the volume to {number}, given it is an integer between 1 and 100
  • Type {text}
    • Types {text} to the screen
